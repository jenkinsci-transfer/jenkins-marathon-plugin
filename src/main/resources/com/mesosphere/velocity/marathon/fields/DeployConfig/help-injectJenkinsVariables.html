This parameter decides whether to inject a few Jenkins Environment Variables into the deployment as Environment Variables.

If this is set to true (checked), then the following Environment Variables will be injected:

  • JENKINS_BUILD_NUMBER = ${BUILD_NUMBER}
  • JENKINS_JOB_NAME = ${JOB_NAME}
  • JENKINS_GIT_COMMIT = ${GIT_COMMIT}
If this is set to false (unchecked), then none of these variables will be injected.
